## <a name="data-asset"></a><span data-ttu-id="d3096-110">Ativos de dados</span><span class="sxs-lookup"><span data-stu-id="d3096-110">Data asset</span></span>

## <a name="structural-metadata"></a><span data-ttu-id="d3096-117">Metadados estruturais</span><span class="sxs-lookup"><span data-stu-id="d3096-117">Structural metadata</span></span>

## <a name="descriptive-metadata"></a><span data-ttu-id="d3096-121">Metadados descritivos</span><span class="sxs-lookup"><span data-stu-id="d3096-121">Descriptive metadata</span></span>

## <a name="preview"></a><span data-ttu-id="d3096-132">Visualização</span><span class="sxs-lookup"><span data-stu-id="d3096-132">Preview</span></span>
<span data-ttu-id="d3096-133">Uma visualização no Catálogo de Dados do Azure é um instantâneo de até 20 registros que podem ser extraídos da fonte de dados durante o registro e armazenados no catálogo com os metadados de ativos de dados.</span><span class="sxs-lookup"><span data-stu-id="d3096-133">A preview in Azure Data Catalog is a snapshot of up to 20 records that can be extracted from the data source during registration, and stored in the catalog with the data asset metadata.</span></span> <span data-ttu-id="d3096-134">A visualização pode ajudar usuários que descobrem um ativo de dados a compreender melhor sua função e finalidade.</span><span class="sxs-lookup"><span data-stu-id="d3096-134">The preview can help users who discover a data asset better understand its function and purpose.</span></span> <span data-ttu-id="d3096-135">Em outras palavras, a visualização de dados de exemplo pode ser mais valiosa do que ver apenas os nomes de coluna e tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="d3096-135">In other words, seeing sample data can be more valuable than seeing just the column names and data types.</span></span>
