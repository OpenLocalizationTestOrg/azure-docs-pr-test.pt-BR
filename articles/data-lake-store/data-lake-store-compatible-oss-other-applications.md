# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-store"></a>Aplicativos de Big Data de software livre que funcionam com o Repositório Azure Data Lake
Este artigo lista os aplicativos de Big Data de software livre que funcionam com o Repositório Azure Data Lake. Para os aplicativos na tabela abaixo, há suporte apenas para as versões disponíveis com a distribuição listada. Para obter informações sobre quais versões desses aplicativos estão disponíveis com o HDInsight, veja [Controle de versão de componente do HDInsight](../hdinsight/hdinsight-component-versioning.md).

| Software livre | Distribuição |
| --- | --- |
| [Apache Sqoop](http://sqoop.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [MapReduce](http://hadoop.apache.org/docs/r1.0.4/mapred_tutorial.html) |HDInsight 3.2, 3.4 e 3.5 |
| [Apache Storm](https://storm.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [O Apache Hive](http://hive.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [HCatalog](https://cwiki.apache.org/confluence/display/Hive/HCatalog) |HDInsight 3.2, 3.4 e 3.5 |
| [O Apache Mahout](http://mahout.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [Apache Pig/Pig Latin](http://pig.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [Apache Oozie](http://oozie.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [O Apache Zookeeper](http://zookeeper.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [Apache Tez](http://tez.apache.org/) |HDInsight 3.2, 3.4 e 3.5 |
| [Apache Spark](http://spark.apache.org/) |HDInsight 3.4 e 3.5 |
