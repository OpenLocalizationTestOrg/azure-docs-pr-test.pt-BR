# <a name="how-tooconfigure-on-premises-encoders-when-using-azure-media-services-toocreate-multi-bitrate-streams"></a>Como tooconfigure local codificadores ao usar fluxos de várias taxas de bits de toocreate de serviços de mídia do Azure
Este tópico lista os codificadores ao vivo do local que você pode usar toocapture seus eventos ao vivo e enviar uma taxa de bits única transmissão ao vivo tooAMS canais (codificação ativa habilitada) para processamento adicional. tópico de saudação também vincula tootutorials que mostram como tooconfigure listados codificadores.

## <a name="elemental-live"></a>Elemental Live
Para obter informações sobre como Olá tooconfigure [Live elementar](http://www.elementaltechnologies.com/products/elemental-live) toosend codificador um fluxo ao vivo de taxa de bits única tooan AMS canal, consulte [Configurando elementar Live](media-services-configure-elemental-live-encoder.md).

## <a name="flash-media-live-encoder"></a>Flash Media Live Encoder
Para obter informações sobre como Olá tooconfigure [Flash Live o codificador de mídia](http://www.adobe.com/products/flash-media-encoder.html) toosend de codificador (FMLE) um fluxo ao vivo de taxa de bits única tooan AMS canal, consulte [FMLE configurando](media-services-configure-fmle-live-encoder.md) .

## <a name="telestream-wirecast"></a>Telestream Wirecast
Para obter informações sobre como Olá tooconfigure [Telestream Wirecast](http://www.telestream.net/wirecast/overview.htm) toosend codificador um fluxo ao vivo de taxa de bits única tooan AMS canal, consulte [Wirecast configurando](media-services-configure-wirecast-live-encoder.md).

## <a name="newtek-tricaster"></a>NewTek TriCaster
Para obter informações sobre como Olá tooconfigure [Tricaster](http://newtek.com/products/tricaster-40.html) toosend codificador um fluxo ao vivo de taxa de bits única tooan AMS canal, consulte [Tricaster configurando](media-services-configure-tricaster-live-encoder.md).
