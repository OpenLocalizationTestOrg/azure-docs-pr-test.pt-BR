# <a name="how-toostop-being-billed-for-azure-remoteapp"></a>Como toostop sendo cobrado Azure RemoteApp
> [!IMPORTANT]
> O Azure RemoteApp será descontinuado até 31 de agosto de 2017. Saudação de leitura [comunicado](https://go.microsoft.com/fwlink/?linkid=821148) para obter detalhes.
> 
> 

Enquanto que você tiver coleções do Azure RemoteApp, mesmo sem qualquer usuário atribuído, você será cobrado pelo RemoteApp. toostop este cobrança, exclua todas as coleções. 

toodelete sua coleção:

1. No Olá portal clássico do Azure, abra a página do RemoteApp hello.
2. Olá coleção você deseja toodelete e, em seguida, clique em **excluir** final Olá Olá página.
3. Repita essa etapa para cada coleção em sua assinatura. 

Pode levar alguns toodelete de minutos cada coleção.
