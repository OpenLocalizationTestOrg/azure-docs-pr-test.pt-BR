## <a name="example-functionimport-for-raw-data-returned-using-post"></a>Exemplo: FunctionImport para dados “Brutos” retornados usando "POST"
Use toocreate POST bruto de dados subordinado novo e seu servidor URL(location) tooupdate parte Olá subordinado no servidor de saudação definido ou URL de retorno.         <d:Title d:Map="" />
        <d:Rights d:Map="" />
