# <a name="azure-toolkit-for-eclipse"></a>Kit de ferramentas do Azure para Eclipse
Olá Kit de ferramentas do Azure para Eclipse fornece modelos e a funcionalidade que você pode usar tooeasily criar, desenvolver, testar e implantar aplicativos do Azure usando o ambiente de desenvolvimento integrado (IDE) do hello Eclipse. Ele é um projeto de código-fonte aberto, cujo código-fonte está disponível em Olá [licença do MIT do site do projeto Olá](https://github.com/microsoft/azure-tools-for-java).

> [!NOTE]
> Também é um kit de ferramentas do Azure para Olá IntelliJ IDE. Para obter mais informações, confira [Kit de Ferramentas do Azure para IntelliJ](azure-toolkit-for-intellij.md).
