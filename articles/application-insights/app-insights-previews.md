# <a name="preview-upcoming-changes-to-azure-application-insights"></a>Visualizar as alterações futuras no Azure Application Insights 

Com frequência, o Application Insights libera novos recursos. Caso deseje ver versões prévias dessas melhorias, inscreva-se na folha Versão Prévia do Application Insights.  A equipe de desenvolvimento disponibiliza versões prévias dos novos recursos de forma limitada antes de liberá-los para todos os usuários. 

A imagem a seguir ilustra como definir suas preferências de versão prévia.

![Folha Versão Prévia do Application Insights](./media/app-insights-preview/preview.png)

## <a name="set-preferences"></a>Definir preferências

Na folha Versão Prévia, selecione uma das opções a seguir para quando as versões prévias são mostradas.

- Sempre: você vê as experiências de Versão Prévia assim que estiverem disponíveis.
- Automático: você vê as experiências de Versão Prévia recomendadas pela Microsoft para sua conta. 
- Nunca: você verá apenas as experiências de Versão Prévia selecionadas.
