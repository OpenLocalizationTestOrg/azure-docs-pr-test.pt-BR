# <a name="service-request-sr-information"></a>Informações de solicitação (SR) de serviço
Fornece Olá informações a seguir quando você abrir uma solicitação de serviço para qualquer serviço do Mobile Engagement:

## <a name="ids-applicable-identifiers-related-tooyour-issue"></a>IDs: Identificadores aplicáveis relacionados ao problema tooyour:
* ID do Aplicativo
* ID da Campanha
* Id do Dispositivo
* Id de Usuário
* Nome de usuário
* Marca de informações do aplicativo

## <a name="errors-applicable-error-information"></a>Erros: Informação de erro aplicável
* Nome da seção de API ou interface de usuário de saudação onde ocorre o problema de saudação
* Texto de qualquer mensagem de erro recebida
* Resultados de testes que você executou da saudação [guias de solução de problemas](http://go.microsoft.com/fwlink/?LinkId=524382)

## <a name="code-applicable-coding-information"></a>Código: Informações de codificação aplicáveis
* A versão do SDK e a plataforma do seu aplicativo (SDK do Android 2.4.1, iOS 1.16.2 etc.)
* Baixe o local do seu aplicativo de produção (ou Olá arquivos APK/TGZ de seu aplicativo de desenvolvimento)
* "AndroidManifest.xml" e/ou qualquer trecho de código do seu aplicativo relacionada tooAzure Mobile Engagement (para solução de problemas avançada)
