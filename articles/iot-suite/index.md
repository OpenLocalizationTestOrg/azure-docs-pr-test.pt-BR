# <a name="iot-suite-documentation"></a>Documentação do IoT Suite

Introdução rápida ao Microsoft Azure IoT Suite. Usar soluções pré-configuradas e acelerar o desenvolvimento de saudação de sua solução de Internet das coisas (IoT). Inclua novos dispositivos e conecte os existentes usando SDKs de dispositivos para diversas plataformas, incluindo Linux, Windows e sistemas operacionais em tempo real. Facilmente dimensionar de apenas alguns toomillions sensores de dispositivos conectados simultaneamente e contar com a disponibilidade global do Azure Olá&mdash;independentemente grande ou pequena seu projeto.
