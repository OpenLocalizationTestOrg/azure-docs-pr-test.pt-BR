<span data-ttu-id="f7a98-106">A engenharia e a seleção de recursos faz parte do Processo de Ciência de Dados de Equipe (TDSP) descrito em [O que é o Processo de Ciência de Dados de Equipe?](data-science-process-overview.md).</span><span class="sxs-lookup"><span data-stu-id="f7a98-106">The engineering and selection of features is one part of the Team Data Science Process (TDSP) outlined in [What is the Team Data Science Process?](data-science-process-overview.md).</span></span> <span data-ttu-id="f7a98-107">A engenharia e a seleção de recursos fazem parte da etapa **Desenvolver recursos** do TDSP.</span><span class="sxs-lookup"><span data-stu-id="f7a98-107">Feature engineering and selection are parts of the **Develop features** step of the TDSP.</span></span>

* <span data-ttu-id="f7a98-108">**engenharia de recursos**: esse processo tenta criar outros recursos relevantes por meio dos recursos brutos existentes nos dados e aumentar a capacidade de previsão do algoritmo de aprendizado.</span><span class="sxs-lookup"><span data-stu-id="f7a98-108">**feature engineering**: This process attempts to create additional relevant features from the existing raw features in the data, and to increase predictive power to the learning algorithm.</span></span>

<span data-ttu-id="f7a98-110">Normalmente, a **engenharia de recursos** é aplicada primeiro para gerar recursos adicionais e, em seguida, a etapa de **seleção de recursos** é feita para eliminar recursos redundantes, altamente correlacionados ou irrelevantes.</span><span class="sxs-lookup"><span data-stu-id="f7a98-110">Normally **feature engineering** is applied first to generate additional features, and then the **feature selection** step is performed to eliminate irrelevant, redundant, or highly correlated features.</span></span>

## <a name="filtering-features-from-your-data---feature-selection"></a><span data-ttu-id="f7a98-111">Filtrando recursos de seus dados - Seleção de recursos</span><span class="sxs-lookup"><span data-stu-id="f7a98-111">Filtering Features from Your Data - Feature Selection</span></span>
<span data-ttu-id="f7a98-112">A seleção de recursos é um processo que geralmente é aplicado à criação de conjuntos de dados de treinamento para tarefas de modelagem de previsão, como tarefas de classificação ou regressão.</span><span class="sxs-lookup"><span data-stu-id="f7a98-112">Feature selection is a process that is commonly applied for the construction of training datasets for predictive modeling tasks such as classification or regression tasks.</span></span> <span data-ttu-id="f7a98-113">O objetivo é selecionar um subconjunto de recursos no conjunto de dados original, reduzindo suas dimensões usando um conjunto mínimo de recursos para representar a quantidade máxima de variação nos dados.</span><span class="sxs-lookup"><span data-stu-id="f7a98-113">The goal is to select a subset of the features from the original dataset that reduce its dimensions by using a minimal set of features to represent the maximum amount of variance in the data.</span></span> <span data-ttu-id="f7a98-114">Esse subconjunto de recursos representa, em seguida, os únicos recursos a serem incluídos para treinar o modelo.</span><span class="sxs-lookup"><span data-stu-id="f7a98-114">This subset of features are, then, the only features to be included to train the model.</span></span> <span data-ttu-id="f7a98-115">A seleção de recursos atende a duas finalidades principais.</span><span class="sxs-lookup"><span data-stu-id="f7a98-115">Feature selection serves two main purposes.</span></span>

* <span data-ttu-id="f7a98-116">Primeiro, a seleção de recursos frequentemente aumenta a precisão de classificação, eliminando recursos irrelevantes, redundantes ou altamente correlacionados.</span><span class="sxs-lookup"><span data-stu-id="f7a98-116">First, feature selection often increases classification accuracy by eliminating irrelevant, redundant, or highly correlated features.</span></span>
* <span data-ttu-id="f7a98-117">Em segundo lugar, ela reduz o número de recursos, o que torna o processo de treinamento de modelo mais eficiente.</span><span class="sxs-lookup"><span data-stu-id="f7a98-117">Second, it decreases the number of features which makes model training process more efficient.</span></span> <span data-ttu-id="f7a98-118">Isso é particularmente importante para tecnologias que sejam dispendiosas de treinar, como máquinas de vetor de suporte.</span><span class="sxs-lookup"><span data-stu-id="f7a98-118">This is particularly important for learners that are expensive to train such as support vector machines.</span></span>

<span data-ttu-id="f7a98-119">Embora a seleção de recursos busque reduzir o número de recursos no conjunto de dados usado para treinar o modelo, ela não costuma ser mencionada usando o termo "redução de dimensionalidade".</span><span class="sxs-lookup"><span data-stu-id="f7a98-119">Although feature selection does seek to reduce the number of features in the dataset used to train the model, it is not usually referred to by the term "dimensionality reduction".</span></span> <span data-ttu-id="f7a98-120">Métodos de seleção de recursos extraem um subconjunto dos recursos originais dos dados sem alterá-los.</span><span class="sxs-lookup"><span data-stu-id="f7a98-120">Feature selection methods extract a subset of original features in the data without changing them.</span></span>  <span data-ttu-id="f7a98-121">Métodos de redução de dimensionalidade empregam recursos de engenharia que podem transformar os recursos originais e, assim, modificá-los.</span><span class="sxs-lookup"><span data-stu-id="f7a98-121">Dimensionality reduction methods employ engineered features that can transform the original features and thus modify them.</span></span> <span data-ttu-id="f7a98-122">Exemplos de métodos de redução de dimensionalidade incluem a Análise de Componente Principal, a análise de correlação canônica e a Decomposição de Valor Singular.</span><span class="sxs-lookup"><span data-stu-id="f7a98-122">Examples of dimensionality reduction methods include Principal Component Analysis, canonical correlation analysis, and Singular Value Decomposition.</span></span>

<span data-ttu-id="f7a98-123">Entre outros, uma categoria amplamente aplicada de métodos de seleção de recursos em um contexto supervisionado é a "seleção de recursos baseada em filtros".</span><span class="sxs-lookup"><span data-stu-id="f7a98-123">Among others, one widely applied category of feature selection methods in a supervised context is called "filter based feature selection".</span></span> <span data-ttu-id="f7a98-124">Ao avaliar a correlação entre cada recurso e o atributo de destino, esses métodos se aplicam a uma medida estatística para atribuir uma pontuação a cada recurso.</span><span class="sxs-lookup"><span data-stu-id="f7a98-124">By evaluating the correlation between each feature and the target attribute, these methods apply a statistical measure to assign a score to each feature.</span></span> <span data-ttu-id="f7a98-125">Em seguida, os recursos são classificados pela pontuação, o que pode ser usado para ajudar a definir o limiar para manter ou eliminar um recurso específico.</span><span class="sxs-lookup"><span data-stu-id="f7a98-125">The features are then ranked by the score, which may be used to help set the threshold for keeping or eliminating a specific feature.</span></span> <span data-ttu-id="f7a98-126">Correlação de Pearson, informações mútuas e o teste de quadrados de Chi pessoa são exemplos das medidas estatísticas usadas nesses métodos.</span><span class="sxs-lookup"><span data-stu-id="f7a98-126">Examples of the statistical measures used in these methods include Person correlation, mutual information, and the Chi squared test.</span></span>
