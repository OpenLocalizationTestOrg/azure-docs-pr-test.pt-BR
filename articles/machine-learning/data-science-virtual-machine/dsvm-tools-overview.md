# <a name="get-to-know-your-data-science-virtual-machine"></a>Conheça sua Máquina Virtual de Ciência de Dados

A Máquina Virtual de Ciência de Dados (**DSVM**) inclui uma ampla lista de Ciência de Dados Popular, Ferramentas de AI/Machine Learning e Plataformas de Dados. Isso foi definido em configurações rigorosamente testadas por Cientistas de Dados e Desenvolvedores da Microsoft, bem como pela comunidade mais ampla de Ciência de Dados, para garantir a estabilidade e a viabilidade geral.

A DSVM também inclui vários exemplos de código e instruções passo a passo na forma de blocos de anotações do Jupyter que podem ser executados localmente. Também há documentação para esses exemplos facilitar a compreensão. 

As ferramentas, as plataformas, os utilitários e as amostras incluídos na DSVM foram organizados nas seguintes seções:

1. **[Aprendizado profundo e estruturas de AI](dsvm-deep-learning-ai-frameworks.md)**
2. **[Ferramentas de Ciência de Dados e ML](dsvm-ml-data-science-tools.md)**
3. **[Linguagens de programação](dsvm-languages.md)**
4. **[Ferramentas de desenvolvimento e IDEs](dsvm-tools-development.md)**
5. **[Ferramentas de ingestão de dados](dsvm-tools-ingestion.md)**
6. **[Ferramentas de exploração e visualização de dados](dsvm-tools-explore-and-visualize.md)**
7. **[Plataformas de Dados](dsvm-data-platforms.md)**
8. **[Exemplos e instruções passo a passo](dsvm-samples-and-walkthroughs.md)**
