# <a name="sprint-2-qfe---december-2017"></a>Sprint 2 QFE - Dezembro de 2017 

#### <a name="version-number-01171115323"></a>Número de versão: 0.1.1711.15323

>Veja como você pode [localizar o número da versão](https://docs.microsoft.com/azure/machine-learning/preview/known-issues-and-troubleshooting-guide).

Bem-vindo à versão QFE (Quick Fix Engineering) da terceira atualização do Azure Machine Learning Workbench. Esta é uma versão secundária. Ela aborda vários problemas de telemetria e ajuda a equipe de produto a entender melhor como o produto está sendo usado. Esse conhecimento pode ser usado em esforços futuros para melhorar a experiência de produto. 

Além disso, há duas atualizações importantes:

- Foi corrigido um bug na preparação de dados que impedia o inspetor de série temporal de exibir em pacotes de preparação de dados.
- Na ferramenta de linha de comando, não é mais necessário ter uma assinatura do Azure para provisionar clusters de ACS de Computação do Machine Learning.
