## <a name="data-retention"></a><span data-ttu-id="5e293-132">Retenção de dados</span><span class="sxs-lookup"><span data-stu-id="5e293-132">Data retention</span></span>
<span data-ttu-id="5e293-133">Os dados sincronizados com o Azure por meio do Enterprise State Roaming serão mantidos indefinidamente, a menos que uma operação de exclusão manual seja executada ou os dados em questão sejam considerados obsoletos.</span><span class="sxs-lookup"><span data-stu-id="5e293-133">Data synced to Azure via Enterprise State Roaming will be retained indefinitely unless a manual delete operation is performed or the data in question is determined to be stale.</span></span> 

<span data-ttu-id="5e293-134">**Exclusão explícita:** os dados são excluídos quando um administrador do Azure exclui um usuário ou um diretório ou um administrador solicita explicitamente que dados devem ser excluídos.</span><span class="sxs-lookup"><span data-stu-id="5e293-134">**Explicit deletion:** The data is deleted when an Azure admin deletes a user or a directory or an admin requests explicitly that data is to be deleted.</span></span>

* <span data-ttu-id="5e293-135">**Exclusão de usuário**: quando um usuário é excluído do Azure AD, os dados de roaming da conta de usuário serão marcados para exclusão e serão excluídos entre 90 a 180 dias.</span><span class="sxs-lookup"><span data-stu-id="5e293-135">**User deletion**: When a user is deleted in Azure AD, the user account roaming data will be marked for deletion and will be deleted between 90 to 180 days.</span></span> 
* <span data-ttu-id="5e293-136">**Exclusão de diretório**: a exclusão de um diretório inteiro no Azure AD é uma operação imediata.</span><span class="sxs-lookup"><span data-stu-id="5e293-136">**Directory deletion**: Deleting an entire directory in Azure AD is an immediate operation.</span></span> <span data-ttu-id="5e293-137">Todos os dados de configurações associados ao diretório serão marcados para exclusão e serão excluídos entre 90 a 180 dias.</span><span class="sxs-lookup"><span data-stu-id="5e293-137">All the settings data associated with that directory will be marked for deletion and will be deleted between 90 to 180 days.</span></span> 
* <span data-ttu-id="5e293-138">**Exclusão mediante solicitação**: se o administrador do Azure AD quiser excluir manualmente os dados de configuração de um usuário específico, poderá criar um [tíquete no suporte do Azure](https://azure.microsoft.com/support/).</span><span class="sxs-lookup"><span data-stu-id="5e293-138">**On request deletion**: If the Azure AD admin wants to manually delete a specific user’s data or settings data, the admin can file a ticket with [Azure support](https://azure.microsoft.com/support/).</span></span> 

<span data-ttu-id="5e293-139">**Exclusão de dados obsoletos**: os dados que não forem acessados para um ano ("o período de retenção") serão tratados como obsoletos e poderão ser excluídos do Azure.</span><span class="sxs-lookup"><span data-stu-id="5e293-139">**Stale data deletion**: Data that has not been accessed for one year (“the retention period”) will be treated as stale and may be deleted from Azure.</span></span> <span data-ttu-id="5e293-140">O período de retenção está sujeito a alterações, mas não será menos de 90 dias.</span><span class="sxs-lookup"><span data-stu-id="5e293-140">The retention period is subject to change but will not be less than 90 days.</span></span> <span data-ttu-id="5e293-141">Os dados obsoletos podem ser um conjunto específico de configurações do Windows/aplicativo ou todas as configurações para um usuário específico.</span><span class="sxs-lookup"><span data-stu-id="5e293-141">The stale data may be a specific set of Windows/application settings or all settings for a user.</span></span> <span data-ttu-id="5e293-142">Por exemplo:</span><span class="sxs-lookup"><span data-stu-id="5e293-142">For example:</span></span>

* <span data-ttu-id="5e293-143">Se nenhum dispositivo acessar uma coleção de configurações específicas (por exemplo, um aplicativo é removido do dispositivo ou um grupo de configurações, como "Tema", está desabilitado para todos os dispositivos do usuário), essa coleção se tornará obsoleta após o período de retenção e poderá ser excluída.</span><span class="sxs-lookup"><span data-stu-id="5e293-143">If no devices access a particular settings collection (e.g., an application is removed from the device, or a settings group such as “Theme” is disabled for all of a user’s devices), then that collection will become stale after the retention period and may be deleted.</span></span> 
* <span data-ttu-id="5e293-144">Se um usuário tiver desativado a sincronização de configurações em todos os dispositivos, nenhum dos dados de configuração será acessado e todos os dados de configuração do usuário se tornarão obsoletos e poderão ser excluídos após o período de retenção.</span><span class="sxs-lookup"><span data-stu-id="5e293-144">If a user has turned off settings sync on all his/her devices, then none of the settings data will be accessed, and all the settings data for that user will become stale and may be deleted after the retention period.</span></span> 
* <span data-ttu-id="5e293-145">Se o administrador de diretório do AD do Azure desativar o Enterprise State Roaming para o diretório inteiro, todos os usuários desse diretório terão a sincronização de configurações interrompida, e todos os dados de configuração de todos os usuários se tornarão obsoletos e poderão ser excluídos após o período de retenção.</span><span class="sxs-lookup"><span data-stu-id="5e293-145">If the Azure AD directory admin turns off Enterprise State Roaming for the entire directory, then all users in that directory will stop syncing settings, and all settings data for all users will become stale and may be deleted after the retention period.</span></span> 

<span data-ttu-id="5e293-146">**Recuperação de dados excluídos**: a política de retenção de dados não é configurável.</span><span class="sxs-lookup"><span data-stu-id="5e293-146">**Deleted data recovery**: The data retention policy is not configurable.</span></span> <span data-ttu-id="5e293-147">Depois que os dados forem excluídos permanentemente, não poderão ser recuperados.</span><span class="sxs-lookup"><span data-stu-id="5e293-147">Once the data has been permanently deleted, it will not be recoverable.</span></span> <span data-ttu-id="5e293-148">No entanto, é importante observar que os dados de configuração só serão excluídos do Azure e não do dispositivo do usuário final.</span><span class="sxs-lookup"><span data-stu-id="5e293-148">However, it’s important to note that the settings data will only be deleted from Azure, not the end-user device.</span></span> <span data-ttu-id="5e293-149">Se qualquer dispositivo se reconectar mais tarde ao serviço Enterprise State Roaming, as configurações serão novamente sincronizadas e armazenadas no Azure.</span><span class="sxs-lookup"><span data-stu-id="5e293-149">If any device later reconnects to the Enterprise State Roaming service, the settings will again be synced and stored in Azure.</span></span>
