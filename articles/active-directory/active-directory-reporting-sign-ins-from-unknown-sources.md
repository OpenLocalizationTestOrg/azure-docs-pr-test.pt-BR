# <a name="sign-ins-from-unknown-sources"></a>Entradas de fontes desconhecidas
Este relatório indica os usuários que entraram com êxito no diretório de tooyour atribuído um endereço IP que foi identificado pela Microsoft como um endereço IP de proxy anônimo (por exemplo, um endereço IP do Tor) cliente. Esses proxies são geralmente usados por usuários que deseja toohide endereço IP do seu computador e podem ser usadas com objetivos mal-intencionados.

Os resultados desse relatório mostrará o número de saudação de vezes que um usuário entrou com êxito no diretório tooyour partir desse endereço e o endereço IP do proxy hello.
