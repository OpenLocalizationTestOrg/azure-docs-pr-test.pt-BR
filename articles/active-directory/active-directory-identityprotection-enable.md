# <a name="enabling-azure-active-directory-identity-protection"></a>Habilitando o Azure Active Directory Identity Protection
O Azure Active Directory Identity Protection é um novo recurso que fornece uma exibição consolidada de atividades de entrada suspeitas e possíveis vulnerabilidades e, com notificações, recomendações de correção e políticas baseadas em risco, ajuda você a proteger sua empresa. 

Olá serviço detecta atividades suspeitas para forçam o usuário final e identidades com privilégios (admin) com base em sinais como bruta ataques, vazarem credenciais, entradas de locais desconhecidos, infectados dispositivos, tooprotect em relação a essas atividades em tempo real. Mais importante, com base em dessas atividades suspeitas, uma severidade de risco do usuário é computada e políticas com base em risco podem ser configuradas e proteger automaticamente identidades de saudação da sua organização. Para obter mais detalhes, confira [Azure Active Directory Identity Protection](active-directory-identityprotection.md).

Este tópico mostra como tooenable proteção de identidade do Azure Active Directory.

## <a name="steps-tooenable-azure-active-directory-identity-protection"></a>Etapas tooenable do Azure Active Directory Identity Protection
1. [Logon](https://ms.portal.azure.com/) tooyour portal do Azure como administrador global. 
2. No portal do Azure de Olá, clique em **Marketplace**.
   
    ![Criar](./media/active-directory-identityprotection-enable/01.png "Criar")
3. Na lista de aplicativos de saudação, clique em **segurança + identidade**.
   
    ![Criar](./media/active-directory-identityprotection-enable/02.png "Criar")
4. Clique em **Azure AD Identity Protection**.
   
    ![Criar](./media/active-directory-identityprotection-enable/03.png "Criar")
5. Em Olá **Azure AD Identity Protection** folha, clique em **criar**.
   
    ![Criar](./media/active-directory-identityprotection-enable/04.png "Criar")
