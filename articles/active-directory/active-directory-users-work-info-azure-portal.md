# <a name="add-or-change-work-information-for-a-user-in-azure-active-directory"></a>Adicionar ou alterar informações de trabalho de um usuário no Azure Active Directory
Este artigo explica como tooadd ou alterar informações de trabalho, como números de telefone ou o departamento de nomes para um usuário no Azure Active Directory (AD do Azure). Para obter informações sobre como adicionar novos usuários em sua organização, consulte [adicionar nova tooAzure de usuários do Active Directory](active-directory-users-create-external-azure-portal.md).

## <a name="toochange-work-information"></a>informações sobre o trabalho toochange
1. Entrar toohello [portal do Azure](https://portal.azure.com) com uma conta que seja um administrador global para o diretório de saudação.
2. Selecione **mais serviços**, digite **usuários e grupos** Olá caixa de texto e, em seguida, selecione **Enter**.

   ![Abrir o gerenciamento de usuários](./media/active-directory-users-work-info-azure-portal/create-users-user-management.png)
3. Em Olá **usuários e grupos** folha, selecione **usuários**.

   ![Folha de usuários de saudação de abertura](./media/active-directory-users-work-info-azure-portal/create-users-open-users-blade.png)
4. Em Olá **usuários e grupos, usuários** folha, selecione um usuário na lista de saudação.
5. Na folha de saudação do usuário selecionado hello, selecione **informações de trabalho**.

    ![Abrindo Informações de Trabalho](./media/active-directory-users-work-info-azure-portal/active-directory-create-users-work-info.png)
6. Adicionar ou alterar informações de trabalho de saudação. Em seguida, na barra de comandos hello, selecione **salvar**.
