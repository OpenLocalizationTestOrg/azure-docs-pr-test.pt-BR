# <a name="create-a-relay-namespace-using-hello-azure-portal"></a>Criar um namespace de retransmissão usando Olá portal do Azure
Um namespace é um contêiner comum para todos os componentes do Azure Relay. Várias retransmissões podem residir em um único namespace e os namespaces geralmente servem como contêineres de aplicativos. Existem atualmente duas maneiras diferentes toocreate um namespace de retransmissão.

1. Portal do Azure (este artigo)
2. Modelos do [Azure Resource Manager](../azure-resource-manager/resource-group-overview.md)
