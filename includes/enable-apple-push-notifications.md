
* [Registrar uma ID para seu aplicativo](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991). Crie uma ID de aplicativo explícita (não uma ID de aplicativo de curinga) e, para **ID do pacote**, usar Olá ID de pacote exata que está em seu projeto de início rápido do Xcode. Também é essencial que você selecione Olá **notificações por Push** opção. 
* Em seguida, [tooprepare para configurar notificações por push](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/AddingCapabilities/AddingCapabilities.html#//apple_ref/doc/uid/TP40012582-CH26-SW6), crie um "Desenvolvimento" ou "Distribuição" certificado SSL.
